# pingpong
Este proyecto es una implementación sencilla del clásico juego Ping Pong, desarrollada en Python utilizando la biblioteca Pygame. El objetivo es controlar una paleta para impedir que la pelota salga de la pantalla, mientras rebota en los bordes y en las paletas.

Características
Desarrollado en Python.
Interfaz gráfica con Pygame.
Movimiento fluido de la pelota y las paletas.
Detección de colisiones entre la pelota, las paletas y los bordes.
Sistema básico de puntuación.
Controles simples mediante el teclado.
Objetivo

El propósito de este proyecto es poner en práctica conceptos fundamentales del desarrollo de videojuegos, como:

Manejo de eventos del teclado.
Animación mediante un bucle principal (game loop).
Detección de colisiones.
Gestión de objetos y movimiento.
Uso de la biblioteca Pygame para crear aplicaciones gráficas.
Tecnologías utilizadas
Python 3
Pygame
Controles
Jugador 1: W (arriba) y S (abajo)
Jugador 2: ↑ (arriba) y ↓ (abajo)
Estado del proyecto

Proyecto educativo orientado al aprendizaje de los fundamentos del desarrollo de videojuegos con Python y Pygame.
